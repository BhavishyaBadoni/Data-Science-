# 🏏 ICC T20 Match Data Analysis

This project analyzes **1417 International T20 cricket matches** using Python and Pandas. The goal is to explore match data and extract useful insights similar to what a **Data Scientist at the International Cricket Council (ICC)** might do.

## 📊 Tasks Performed

* Renamed dataset columns to more readable names (e.g., `meta.created → created_date`)
* Identified the **top 3 venues** hosting the most matches
* Found the **pair of teams that played the most matches against each other**
* Calculated the **top 5 teams by win percentage**
* Created a **function to generate match scorecards**, showing:

  * Top 4 scorers of the batting team
  * Top 4 bowlers of the opposing team

## ⚙️ Technologies Used

* Python
* Pandas
* Jupyter Notebook

This project demonstrates **data cleaning, exploratory data analysis, and cricket performance insights** using real match data.
